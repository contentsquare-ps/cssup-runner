# cssup-runner
Cssup management


## Install
`npm i -g cssup-runner`

## Usage
`cssup-runner`
`cssup-runner run`
Check if cssup exists
* yes - check version
* no - install
Run cssup

`cssup-runner help`
show this

`cssup-runner remove`
removes the cssup folder

`cssup-runner version`
get current version
